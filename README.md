# Justinian-LOD

This repository contains the materials developed for the project  **“Iustinianus LOD”** , created within the course *Information Science and Cultural Heritage* at the University of Bologna (a.y. 2024–2025), taught by **Marilena Daquino** and  **Francesca Tomasi** .

The project focuses on **Emperor Justinian I (527–565 CE)** and aims to explore his historical and cultural legacy through the creation of a  **Linked Open Data dataset** . By collecting and modeling a selection of cultural heritage items connected to Justinian’s reign, the project builds a semantic network that links artworks, architectural monuments, historical texts, and modern cultural representations.

The dataset has been structured according to Semantic Web principles, transforming descriptive metadata into **RDF triples** and linking entities such as people, places, events, and concepts through established ontologies and vocabularies.
